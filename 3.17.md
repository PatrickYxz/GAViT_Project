### 1. 搭建 GAViT 核心代码结构

从零创建了 `models/` 目录，包含以下模块：

| 文件 | 内容 |
|------|------|
| `models/swin_backbone.py` | 封装 Swin-T，去掉分类头，输出 `(B, 49, 768)` tokens |

## 修复的 Bug

| 问题 | 原因 | 解决方式 |
|------|------|---------|
| `sbatch: DOS line breaks` | Windows 创建的 `.sh` 文件行尾为 CRLF | `sed -i 's/\r//'` + `.gitattributes` |
| `ModuleNotFoundError: torch_geometric`（第一次）| `models/__init__.py` eager import GAViT | 清空 `__init__.py`，改为各脚本自行 import |
| `ModuleNotFoundError: torch`（第二次）| sbatch 脚本 conda 激活顺序错误 | 改为 `source activate` → `conda activate gavit` |
| `git pull` 无 tracking | 服务器分支为 `master`，未追踪 `origin/main` | `git branch --set-upstream-to=origin/main master` |
